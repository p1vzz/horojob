# Career Insights Platform (US Market)

## Overview

This project is a career recommendation platform focused on the United States job market. The goal is to help users explore career paths using reliable labor market data combined with personalized recommendation algorithms.

The platform integrates data such as:

* Salary ranges
* Job outlook and demand
* Required skills and qualifications

This information is used to generate structured and user-friendly career insights.

## Purpose

The application is designed to support users in making informed career decisions by combining data-driven insights with personalized recommendations.

The primary audience includes individuals exploring career opportunities in the U.S. market.

## Data Sources

The platform uses publicly available labor market data, including data provided by CareerOneStop (U.S. Department of Labor).

All data usage follows the official terms of use, including proper attribution and compliance with usage guidelines.

## Project Status

The project is currently in active development. Core features include:

* Career data integration
* Recommendation engine
* User-facing insights interface

Additional features and improvements are being implemented.

## Compliance

We are committed to:

* Providing proper attribution to data sources
* Not redistributing data as a standalone product
* Using the data only as part of a value-added application
